# rogue-orchestrator-widgets

A collection of newsletter-embeddable widgets. Hosted via GitHub Pages.

## Widgets

- **token-cost** — `/token-cost/`

## Embedding

Once GitHub Pages is enabled, embed in a newsletter via iframe:

```html
<iframe
  src="https://archaeus-ai.github.io/rogue-orchestrator-widgets/token-cost/"
  width="100%" height="600" frameborder="0"></iframe>
```

## Enable GitHub Pages

Settings → Pages → Source: **Deploy from a branch** → Branch: `main` / `/ (root)` → Save.
